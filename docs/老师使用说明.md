# Office Copilot Lite 使用说明

> 在 Excel / Word / PowerPoint 里用上 AI 助手,让它帮你**填表、写文档、做 PPT**。

---

## 📦 这是什么

一个**安装在你电脑上**的 Office 加载项。装好后,Excel/Word/PPT 顶部菜单会多出一个 "Office Copilot Lite" 按钮,点开右侧会出现 AI 聊天面板。

你用普通话给它说话,它**真的去操作单元格 / 段落 / 幻灯片**——不是只发文字给你复制粘贴。

### 真实使用场景

- 📊 "把第二个 sheet 的学生信息按班级分组汇总到当前 sheet"
- 🧮 "在 G 列写公式,根据 D 列学号判断班级"
- 📝 "根据 Sheet1 的数据写一份会议纪要,5 段,正式书面语"
- 🎬 "做一份 10 页项目汇报 PPT,主题是 AI 助手"
- ➕ "把这个 docx 里的学生姓名/学号/电话填进表里,留空的字段不动"

## 📂 zip 包里有什么(各文件功能速查)

解压 zip 后看到这些文件,**你只需要双击 `install.bat`**,其他都是兜底用的。

### 🚀 安装入口(3 选 1,效果一样,**优先用第一个**)

| 文件 | 何时用 |
|---|---|
| **`install.bat`** ⭐ | 主入口,99% 老师机器双击就能跑 |
| `install.vbs` | 兜底——如果双击 .bat 弹"选择应用"对话框(说明 .bat 关联被某编辑器改坏了),改双击这个 |
| `双击安装.lnk` | 终极兜底——以上都不行时用这个,绝对能跑 |

### 🗑 卸载入口(2 选 1)

| 文件 | 用途 |
|---|---|
| `uninstall.bat` | 卸载主入口 |
| `uninstall.vbs` | 兜底卸载(同上) |

### 🛠 日常工具(装完后桌面会有快捷方式,不需要进解压目录找)

| 文件 | 用途 | 何时用 |
|---|---|---|
| `重启服务.bat` | 重启后台 Helper 进程 | 加载项突然连不上 / 面板转圈 / 报错 |

桌面装完后会看到 2 个快捷方式,这是日常常用入口:
- **"重启服务(Office Copilot Lite)"** — 出问题时双击
- **"卸载 Office Copilot Lite"** — 不想用了双击

### 📁 其他(用户**不需要碰**)

- `bin/helper.exe` — 后台服务程序
- `scripts/` — 安装脚本(PowerShell)
- `docs/` — 本说明文档

## 🛠 安装(只需 30 秒,1 次双击,无需输入)

### 系统要求

| 项 | 要求 |
|---|---|
| 操作系统 | Windows 10/11 |
| Office 版本 | **Microsoft 365** 或 **Office LTSC 2024** |
| ⚠️ Office LTSC 2021 / Office 2019 / WPS | ❌ 不支持(加载项识别不到) |

### 步骤

1. **解压**下载到的 zip 到任意位置(推荐 `C:\OfficeCopilotLite`)
2. **双击** `install.bat`
   - 如果双击没反应或弹"选择应用",改双击 **`install.vbs`** 或 **`双击安装.lnk`**
3. 弹出 UAC 确认 → 点 **"是"**
4. 等 ~30 秒,看到 **"✅ 安装成功"** 对话框即完成
5. 桌面会自动出现 2 个快捷方式:
   - 重启服务(Office Copilot Lite)
   - 卸载 Office Copilot Lite

### 在 Excel/Word/PPT 里启用 + 配置(只需一次)

1. 打开 **Excel**(或 Word / PowerPoint)
2. 顶部菜单 **「开始」** → 找到 **「加载项」** 按钮
3. 弹窗顶部切到 **「共享文件夹」** 标签
4. 看到 **"Office Copilot Lite"** → 双击 / 点添加
5. 等 5 秒,右侧出现 AI 聊天面板

**首次打开会让你填 3 个字段**:

| 字段 | 填什么 |
|---|---|
| **Gateway URL** | `https://api.deepseek.com/anthropic`(DeepSeek 官方,推荐) |
| **API Key** | 你的 DeepSeek API Key(在 https://platform.deepseek.com 申请,以 `sk-` 开头) |
| **Auth Header** | `X-Api-Key` |

点保存就能用了。**所有 Office 应用共用同一份配置**。

## 💬 怎么用

在右侧 AI 面板的输入框里**用普通话**给它说话:

| 你说 | 它做 |
|---|---|
| "在 A1:E1 写表头:姓名/班级/学号/手机号/户籍,自动调列宽" | 真的去写,而不是让你手动 |
| "把 B 列的日期格式改成 yyyy-MM-dd" | 选中 B 列改格式 |
| "G 列用 VLOOKUP 查 F 列对应班级" | 真的写公式 |
| "把上传的 docx 里 28 个学生信息填进表格" | 解析文档 + 写入 |

支持上传文件(docx/xlsx/pdf):**直接拖到聊天框**,它会读取内容。

## ⚙️ 常见问题

### Q1:双击 install.bat 没反应 / 弹"选择应用"对话框

说明你电脑的 `.bat` 文件关联被某个编辑器(VS Code / Notepad++ / Sublime 等)改了。

解决:改双击 **`install.vbs`** 或 **`双击安装.lnk`**(都是同一个安装流程,效果一样)。

### Q2:Excel 里看不到加载项

1. **任务管理器**关掉所有 EXCEL.EXE 进程
2. 重开 Excel
3. 文件 → 选项 → 加载项 → 管理"我的加载项",看 "Office Copilot Lite" 在不在
4. 还不行 → 双击桌面 **"重启服务"** 快捷方式

### Q3:聊天面板提示连接失败 / 长时间转圈

LLM 网关临时挂了,或 Key 不对。尝试:

1. 等 5 分钟重发
2. 在 Excel AI 面板里改 URL/Key(直接在面板里改,不需要重启)
3. 双击桌面 **"重启服务"**

### Q4:换 Key

直接在 Excel 加载项右侧 AI 面板里改 URL/Key 即可,**不需要重启电脑或服务**。

### Q5:想卸载

桌面 **"卸载 Office Copilot Lite"** → 双击 → 按提示。

如果桌面快捷方式无效,直接到解压目录双击 **`uninstall.bat`**(或 `uninstall.vbs` 兜底)。

会清理:
- 登录启动任务
- Excel 信任目录
- 网络共享
- 安装目录
- (可选)自签证书

### Q6:出问题怎么办

把这个文件发给作者排查:

```
%LOCALAPPDATA%\OfficeCopilotLite\install.log
```

(在文件资源管理器地址栏里粘贴上面这行回车,会直接定位到文件)

## 🔒 隐私 & 安全

- **不收集任何数据**:本机 `helper.exe` 只 serve 加载项 webview,你的对话内容**直接从浏览器 fetch 到你配置的 LLM 网关**,不经过任何中间服务器
- **API Key 只存加载项内部存储**:不写硬盘文件,不上传任何地方
- **服务只监听本地端口 18765**:不对外网开放
- **自签证书域名锁定**:`localhost` 专用,作用域只在当前用户
